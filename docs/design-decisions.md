# Decisiones de Diseño — Praxi

## Stack UI (Issue #10)

**Decisión:** shadcn/ui + Tailwind CSS

### Componentes
- **shadcn/ui** — componentes copiados al proyecto, no dependencia externa
- Basado en Radix UI (accesibilidad) + Tailwind (estilos)

### Estilos
- **Tailwind CSS** — utility-first, estándar con Next.js

### Recursos
- [shadcn/ui](https://ui.shadcn.com) — librería base
- [21st.dev](https://21st.dev) — componentes y bloques adicionales para shadcn
- Templates de dashboard admin basados en shadcn para referencia

### Criterios
- Aspecto moderno y minimalista ✅
- Compatible con Next.js ✅
- Componentes de gestión (tablas, forms, calendarios, modales) ✅
- Personalizable (el código es nuestro) ✅
- Comunidad activa y documentación ✅
